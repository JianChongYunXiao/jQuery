JQ是一个类（也是一个普通对象）：函数的两种角色，JQ是一个类库提供了很多方法，其中这些方法由两部分：
  1. 放到JQ原型上的（jQuery.fn / jQuery.prototype），这里面的方法是供JQ实例调取使用的
  2. 把JQ当作一个普通对象，在对象上设置一些私有属性和方法，这类方法以后用的时候直接jQuery.xxx()执行即可

六、 自定义事件

JQ的事件，都是事件绑定的 
JQ的所有事件都是on()这个方法扩展出来的 
注意：在事件套事件的时候，容易出问题（重复绑定的问题） 
解决：在事件套事件的时候，要把上一次的事件清掉，使用off() 
off（可以放精确的范围）比如：具体的事件名、事件名的class

（1）收集器（订阅模式） add

目的就是为了收集事件和事件函数

（2）触发器（发布模式）trigger

当某个条件成立之后执行对应事件的事件函数

（3）解除器 off

销毁对应事件的事件函数
//--------------------这是JS自定义事件！！！
//--------------------收集器=> 订阅模式
function add(ele, evName, fn) {
ele.diy = ele.diy || {}; //给对象绑定一个diy属性，值为对象，如果之前绑定过了，就用之前的，如果从来没有绑定过，就用空对象
ele.diy[evName] = ele.diy[evName] || []; //看看之前这个对象有没有同样的属性，有就用之前的，没有就用空数组
ele.diy[evName].push(fn);
}
//--------------------触发器=> 发布模式
function trigger(ele, evName) {
if (!ele.diy[evName]) return null; //如果元素没有绑定过这个事件，返回null
ele.diy[evName].forEach(fn => {
fn.call(ele); // 此时this是btn
})
}
//--------------------解除器
function off(ele, evName, fnName) {
if (!ele.diy[evName]) return null;
for (let i = 0; i < ele.diy[evName].length; i++) {
//循环对象种对应的事件数组，找到数组中和第三个参数的函数名一致的函数，删除
//给对象加上属性名是索引的自定义属性，再添加上length，这样可以使用for循环循环length（像数组似的）就可以循环绑定的自定义属性。
if (ele.diy[evName][i].name === fnName.name) {
ele.diy[evName].splice(i, 1);
i--;
}
}
}
// 点了点击按钮会弹出 '不要点我' '你咋还点'
// 点击删除按钮，在点 点击按钮 弹出 '不要点我' 不会再出现 '你咋还点'
add(btn, '点击', function () {alert('不要点我')})
add(btn, '点击', fn);
function fn() {alert('你咋还点')}
btn2.onclick = function () {off(btn, '点击', fn);}
btn.onmousedown = function (ev) {
let x = ev.pageX;
let y = ev.pageY;
btn.onmouseup = function () {
if (Math.abs(ev.pageX - x) < 5 && Math.abs(ev.pageY - y) < 5) {
trigger(btn, '点击')
}
}
}

（4）运动

jQuery中的运动是基于animate()
  • animate()：用于创建自定义动画的函数 
    • 注意：所有指定的属性必须用驼峰命名，比如用marginLeft代替margin-left
  • stop([queue],[clearQueue],[jumpToEnd])：停止所有在指定元素上正在运行的动画。 
    • queue：用来停止动画的队列名称
    • clearQueue：如果设置成true，则清空队列。可以立即结束动画
    • jumpToEnd：如果设置成true，则完成队列。可以立即完成动画
  • show([speed,[easing],[fn]]) ：显示隐藏的匹配元素。如果选择的元素是可见的，这个方法将不会改变任何东西。无论这个元素是通过hide()方法隐藏的还是在CSS里设置了display:none;，这个方法都将有效。 
    • speed：表示动画时长的毫秒数值(如：1000)
    • easing:(Optional) ：用来指定切换效果，默认是”swing”，可用参数”linear”
    • fn：在动画完成时执行的函数，每个元素执行一次。
  • hide([speed,[easing],[fn]])：隐藏显示的元素 如果选择的元素是隐藏的，这个方法将不会改变任何东西。
  • toggle([speed],[easing],[fn])：用于绑定两个或多个事件处理器函数，以响应被选元素的轮流的 click 事件。如果元素是可见的，切换为隐藏的；如果元素是隐藏的，切换为可见的。
  • animate(params,[speed],[easing],[fn])：创建自定义动画的函数。
    • params：一组包含作为动画属性和终值的样式属性和及其值的集合
  • delay(duration,[queueName])：设用于将队列中的函数延时执行。他既可以推迟动画队列的执行，也可以用于自定义队列。 
    • duration：延时时间，单位：毫秒
    • queueName：队列名词，默认是Fx，动画队列。
$('#btn').click(function () {
$('#box').animate({

}, 100).delay(1000);
$('#box').animate({
